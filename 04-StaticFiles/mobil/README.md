# Kliens adatok elemzése

A mobil eszközök sávszélessége jellemzően alacsonyabb az asztali alkalmazásokhoz képest, ezért letölthető fájlokat célszerű optimalizálni.

## Ajánlott projekt struktúra

```text
your-project/
├── public/
│   ├── index.html
│   ├── mobile.html (opcionális)
│   ├── css/
│   │   ├── style.css
│   │   └── mobile.css
│   ├── js/
│   │   ├── app.js
│   │   └── mobile.js
│   ├── images/
│   │   ├── logo.png
│   │   └── mobile/
│   └── assets/
├── package.json
└── server.js
```
